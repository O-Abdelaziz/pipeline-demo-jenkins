# Flask CI/CD Pipeline Demo

Minimal Flask app with a Jenkins pipeline, Docker packaging, and pytest coverage.

Repository: https://github.com/O-Abdelaziz/pipeline-demo-jenkins.git

Includes:
- Flask app UI in `app/`
- Tests in `tests/`
- Jenkins pipeline in `Jenkinsfile`
- Docker image and Compose deployment
- Captured screenshots in `screenshots/`
